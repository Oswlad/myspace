  • 兩百字內的個人簡介

    目前是在職中的前端工程師，主要負責公司官網程式碼維護，及公司產品使用的網頁，使用的程式語言以html css javascript為主，預計於今年轉職，在轉職前希望能加強自己前端技術及學習更多後端相關的技術，包含資料庫，資料串接，以利後續轉職的需要

  • 為什麼想成為工程師 ( 請以自我期許帶入：前端、後端、或全端工程師 )

    寫程式是興趣之一，起初覺得網頁很神奇，寫一行 HTML 就可以馬上看到成果，所以就萌生了想學習如何產生網頁的契機，入職場後才知道原來一個網頁的產生其實是很多東西組成，包含前端的 UI/UX， 及後端的資料庫，資料如何串接，資料庫如何規畫，網頁的效能如何提升．．．等等，這些都是可以深入學習課題。

  • 為了成為工程師，曾經做過什麼努力？

    初入職場時，發現公司需要的與自己所學其實有一段差距，於是開始利用下班時間進修，內容包含重新學習 HTML、CSS、jQuery、還有Amos的金魚都能懂的網頁教學系列， 不找還好，一找發現線上的資源其實超級多，中文、英文任選，youtube上也是個很好的來源，線上資源平台包含 Udemy、Hohow、臺大開放課程...等等，如果有遇到解決不了的問題， 我會先上 stackoverflow 上搜尋是否有人有跟我一樣遇過類似的問題，通常會很容易在上面找到類似的問題及解答。

  • 對於工程師的職業想像是什麼？
